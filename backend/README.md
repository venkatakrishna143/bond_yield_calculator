# Bond Yield Calculator

Full-stack Bond Yield Calculator built using:

- Backend: NestJS (TypeScript)
- Frontend: React (TypeScript)

## Features

- Current Yield
- Yield to Maturity (YTM)
- Premium / Discount Detection
- Cash Flow Schedule
- Annual & Semi-Annual Frequency

## How to Run

### Backend
npm install
npm run start:dev

### Frontend
npm install
npm start
